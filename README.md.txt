Uma calculadora de IMC simples e responsiva, desenvolvida para fins de estudo e portfólio, utilizando tecnologias web fundamentais. O projeto calcula o Índice de Massa Corporal com base no peso e altura fornecidos pelo usuário e exibe o resultado junto com a classificação correspondente da OMS
## Funcionalidades

- Entrada de dados para peso (em kg) e altura (em metros).
- Cálculo em tempo real com o clique de um botão.
- Exibição do valor de IMC com duas casas decimais.
- Classificação do resultado (Abaixo do peso, Peso normal, Sobrepeso, etc.).
- Interface com cores dinâmicas que mudam de acordo com o resultado para feedback visual imediato.
- Design totalmente responsivo, adaptável para desktops e dispositivos móveis.

## Tecnologias Utilizadas

Este projeto foi construído com:

- HTML5: para a estrutura e semântica do conteúdo.
- CSS3: para estilização, layout (Flexbox) e responsividade.
- JavaScript: para a lógica de programação, manipulação do DOM e cálculo do IMC.
